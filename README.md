# React + TypeScript + Vite


<p>Este proyecto consiste en el desarrollo de una aplicación web (SPA) orientada a la gestión de frases. Su objetivo principal es aplicar y consolidar la creación de interfaces dinámicas, la estructuración de componentes reutilizables y la integración con servicios externos mediante una API REST.
Para construir una base moderna, eficiente y escalable, el proyecto se ha desarrollado utilizando React, TypeScript y Vite. Además, se han implementado herramientas y buenas prácticas propias de entornos profesionales:
Navegación y Seguridad: Enrutado y protección de rutas privadas utilizando React Router.
Gestión de Estado: Control global de la autenticación mediante Context API.
Formularios: Validación robusta y manejo eficiente de datos con React Hook Form y Zod.
Rendimiento: Optimización de la aplicación mediante carga diferida (lazy loading) con Suspense.
El resultado es una plataforma con un diseño totalmente responsive que ofrece una experiencia de usuario fluida y coherente. Incluye funcionalidades clave como autenticación (login/logout), operaciones CRUD completas sobre las frases y un sistema de marcadores de favoritos.</p>

​​​​​​​
<p>This project involves the development of a Single Page Application (SPA) designed for quote management. Its primary goal is to put into practice the creation of dynamic interfaces, the structuring of reusable components, and the integration of external services via a REST API.
To build a modern, efficient, and scalable foundation, the project was developed using React, TypeScript, and Vite. Furthermore, industry-standard tools and best practices were implemented throughout the development process:
Routing & Security: Seamless navigation and private route protection using React Router.
State Management: Global authentication control handled by the Context API.
Forms: Robust data validation and efficient form handling using React Hook Form and Zod.
Performance: Application optimization through lazy loading components with Suspense.
The result is a fully responsive platform that delivers a smooth and cohesive User Experience (UX). It includes key features such as user authentication (login/logout), full CRUD operations for quotes, and a favorites bookmarking system.</p>
